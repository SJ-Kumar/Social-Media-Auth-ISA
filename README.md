## Setup process

Run the below command in the root directory to install all required packages for the backend server:

```
npm install
```

Create FireBase Account and get the required Secret Keys

```
apiKey: "",
authDomain: "",
projectId: "",
storageBucket: "",
messagingSenderId: "",
appId: "",
measurementId: ""

```

## Running process

Run the below command in the root directory:

```
npm start

```
